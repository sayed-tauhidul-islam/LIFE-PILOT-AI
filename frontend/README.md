# Frontend - Life Pilot AI Agent

React frontend application for the Life Pilot AI Agent.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will run on `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Technologies

- React 18
- Vite
- Tailwind CSS
- Axios for API calls

## Environment Variables

Create a `.env` file if you need to configure API endpoints:

```
VITE_API_URL=http://localhost:5000
```
